## Project Description
ImeConverter2025 is a Windows Forms application developed in C# that allows users to convert temperatures between Celsius and Fahrenheit. The project demonstrates core WinForms concepts, event-driven programming, form navigation, and clean project structure as required for the GLA.

---

## Features
- Convert temperatures between:
  - Celsius → Fahrenheit
  - Fahrenheit → Celsius
- User-friendly Windows Forms interface
- Separate About form for application information
- Input validation through UI controls
- Clear and readable layout
